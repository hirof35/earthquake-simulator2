# 🗺️ Visual Earthquake Simulator (TypeScript & Canvas)
<img width="1197" height="816" alt="スクリーンショット 2026-06-14 080442" src="https://github.com/user-attachments/assets/586a961e-232d-46e0-b477-2b296aa98af8" />

TypeScript と HTML5 Canvas API を使用して構築された、ブラウザで動作する2次元の**視覚的地震シミュレーター**です。
地震波（P波・S波）の伝播速度の違いや、震度（マグニチュード）に応じた構造物の動的な倒壊挙動を視覚的に体験・理解することができます。

---

## 🚀 主な機能と特徴

* **地震波の分離再現 (P波・S波)**
    * 地震発生時、まず伝播速度の速い**初期微動（P波：青い円）**が広がります。
    * 遅れて、強い揺れを伴う**主要動（S波：赤い円）**が広がります。
* **動的な物理演算（簡易版）**
    * 各建築物（ビル）は、**S波がその地点に到達した瞬間**から激しく揺れ始めます。
    * ビルの高さ（固有周期・耐震基準のシミュレート）に応じて揺れへの耐性が異なり、限界を超えて傾くと完全に倒壊します。
* **インタラクティブな操作**
    * スライダーにより、マグニチュード（1.0 〜 9.0）を自由に変更可能。
    * エネルギー計算式（$2^{M-3}$）をベースにしているため、マグニチュードの大きさに応じて指数関数的に揺れの激しさと被害規模が増大します。
* **オブジェクト指向・モジュール設計**
    * TypeScript のクラスを活用し、シミュレーター全体を管理する `Simulator`、個々の建物を制御する `Building` にクリーンに分割されています。

## 📂 フォルダ構成

```text
earthquake-simulator/
├── dist/                  # コンパイルされたJavaScriptの出力先
├── src/                   # TypeScript ソースコード
│   ├── Building.ts        # ビルの物理演算・描画ロジック
│   ├── Simulator.ts       # 地震波の計算・キャンバス全体の管理
│   └── main.ts            # エントリーポイント（UIイベントとの紐付け）
├── index.html             # フロントエンド画面（HTML/CSS）
├── package.json           # プロジェクトの依存関係・スクリプト定義
└── tsconfig.json          # TypeScript コンパイル設定
🛠️ セットアップと実行方法
1. 依存関係のインストール
プロジェクトのルートディレクトリで以下のコマンドを実行し、TypeScript などの開発環境を構築します。

Bash
npm install
2. TypeScript のコンパイル
ソースコード（.ts）をブラウザが理解できる JavaScript（.js）に変換します。

Bash
# 1回のみコンパイルする場合
npm run build

# ファイルの変更を監視して自動コンパイルする場合
npm run watch
3. ローカルサーバーでの起動
ブラウザのセキュリティ制限（CORSポリシー）により、index.html を直接ダブルクリックで開くとエラーになる場合があります。以下のいずれかの方法でローカルサーバーを起動して閲覧してください。

方法 A: VS Code の拡張機能「Live Server」で index.html を開く。

方法 B: ターミナルで簡易サーバーを起動する。

Bash
npx serve .
起動後、画面に表示される http://localhost:3000 などのURLにブラウザでアクセスします。

💡 今後の拡張アイデア（ロードマップ）
このプロジェクトはモジュール化されているため、以下のような機能拡張が簡単に行えます。

ビルごとに「耐震・制震構造」のパラメーターを追加し、補強による効果を比較する。

液状化現象を再現するため、地面（地盤）のグラフィックや挙動を変化させる。

横揺れだけでなく、縦揺れ（直下型地震）のパターンを追加する。
